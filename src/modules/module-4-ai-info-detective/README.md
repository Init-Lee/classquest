# Module 4 — AI 信息侦探

模块 4 是七年级收束模块，目标是围绕新闻与图片中的 AI 痕迹判断，完成题卡创作、教师审核、匿名试答、快速评分和画廊发布。

## 当前状态

本轮架构重构只创建占位，不实现具体课程内容。

已创建：

- `module.config.ts`
- `routes.tsx`
- `pages/Module4HomePage.tsx`
- `README.md`
- `FILE-STRUCTURE.md`

## 后续课程结构

1. 框架发布与终版样例分析
2. 素材收集与合规筛查
3. 题卡创作与解释撰写
4. 同伴评审与终稿优化
5. 网页试答与反馈优化
6. 题库发布与可信反思

## 开发策略

- Phase A：本地前端流程。
- Phase B：mock API 提交、审核状态、试答、评分、画廊。
- Phase C：接入 FastAPI。
- Phase D：统计重算与频段展示。

## 边界规则

模块 4 业务组件留在本目录内。即使与模块 3 视觉相似，也不要直接 import 模块 3 业务代码。
