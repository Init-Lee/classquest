# Module 4 — AI 信息辨识员

模块 4 是七年级收束模块，目标是围绕新闻与图片中的 AI 痕迹判断，完成题卡创作、教师审核、匿名试答、快速评分和画廊发布。

## 当前状态

课时 1「框架发布与样例拆解」已进入本地前端实现阶段，提供独立 Module4Provider、IndexedDB 本地档案、继续学习包、阶段快照和教师演示模式；继续学习包命名为 `模块4_姓名_当前进度_日期.json`，阶段快照按第 1～5 关顺序导出并内置提交用样式。顶栏「课时 1～6」胶囊进度条与课内「第 N 关」流程条与模块三对齐。第 1 关任务发布采用「固定关卡进度条 + 下方纵向吸附分屏滚动」与出血背景分段（配图置于 `lessons/lesson-1/assets/`）。第 2 关样例观察分屏完成新闻类/图片类的观察判断与解析核验；第 3 关用田字型结构配对拆解题卡四部分；第 4 关以说明书形式拆解完整题卡模板（底部单次总确认）；第 5 关作为课时 1 出口任务单，填写候选素材计划，并在左右分栏阅读「避免使用素材」与「出口确认」后**一次性勾选**完成课时。

首页建档与模块三一致：姓名、班级（初一（1）班～初一（12）班）、班学号（左侧班级序号只读 + 右侧学号后两位 01～50，合成四位存档）；无档案不可直接进入课时 1（直达 `/module/4/lesson/1/...` 会退回 `/module/4`）。

## 后续课程结构

1. 框架发布与终版样例分析
2. 素材收集与合规筛查
3. 题卡创作与解释撰写
4. 同伴评审与终稿优化
5. 网页试答与反馈优化
6. 题库发布与可信反思

## 开发策略

- Phase C：接入 FastAPI。

## 教师模式

教师入口位于 `/module/4` 首页。进入后可使用顶部演示横幅切换「空白态 / 进行中 / 已完成」三档演示数据，Guard 会放行，保存只写内存演示档案。
