# Lesson 3 — 题目卡 V1 制作与解析填写

本目录实现模块四课时 3 的 dev 分支前端流程。课时 3 只完成两张 V1 初稿：新闻题卡 V1 与图片题卡 V1。

## 流程

1. V1 规则确认：明确本课只做初稿。
2. 新闻题卡编辑器：从课时 2 新闻素材复制快照，在**单屏编辑工作台**（四 Tab + 右侧反馈面板）填写题干、答案、解析与核验入口。
3. 图片题卡编辑器：从课时 2 图片素材复制快照，复用同一工作台组件填写四部分。
4. 双卡自测与 V1 保存：分别试答新闻/图片题卡，查看反馈并确认体验；两张均确认且四部分齐全后保存为 `ready_for_lesson4`。Step2/Step3 编辑后会失效对应自测记录，需重新试答确认。

## 边界

本课不实现同伴互审、教师审核、V2、难度标注、正式入库、网页试答、Q 分数或频率调节。出口文案只提示下一课进入互审与 V2 优化。

## AI 自检

组件只调用模块 4 API adapter：`api/lesson3-ai-review.adapter.ts`。默认 mock；设置 `VITE_MODULE4_LESSON3_AI_REVIEW_MODE=http` 后调用 `/api/v1/module4/lesson3/ai-review`。自检失败不阻止保存 V1。
