# Module 4 Lesson 2 — 素材搜集与合规初筛

课时 2 把课时 1 的素材准备计划推进为两份可进入下一课的素材体检记录：

1. 新闻素材体检记录。
2. 图片素材体检记录。

它为课时 3 的题卡创作提供干净输入，但本课不制作完整题目卡。

## 教学边界

课时 2 会做：

- 上传、替换、补全新闻截图与图片素材。
- 通过说明页和校准案例建立四项素材体检标准。
- 第 2 关标准说明首次学习时逐项解锁，并仅记录每项停留时间作为过程数据。
- 第 2 关标准练习记录每次选择事件，用于还原每题尝试次数、修正次数和第几次答对。
- 在新闻和图片工作台开头，用四关标准复判自己的真实素材。
- 保留第 1 关三个素材快判答案，二次进入时用于回顾而不是重复作答。
- 完成来源记录格式检查。
- 完成类型、合规、判断价值三项自检。
- 写 1 条初步疑点提示。
- 记录 1 条同伴或自我交流提醒。
- 自动生成 QuickCheck。

课时 2 不做：

- 完整题目卡创作。
- 选项设置或正确答案选择。
- 最终解释撰写。
- OCR、真实来源核验或 AI 鉴定。
- 后端提交或教师评分。

## 五关流程

```text
1. 素材不等于题目
2. 四关体检标准：标准说明页与四关标准校准
3. 新闻素材工作台：按截图、来源、复核、记录、完成检查五段式补全新闻素材
4. 图片素材工作台：按图片、来源、复核、记录、完成检查五段式补全图片素材
5. 我的素材体检报告
```

核心推进关系：

```text
快速判断 → 建立标准 → 个人素材工作台补全与复核 → 体检报告
```

## 资产处理

- 新闻素材主体是新闻截图，不是手打标题与正文。
- 图片素材主体是单张静态图片。
- 第 1 关的新闻/图片准备状态是一次性登记：学生选择后锁定状态选项，只允许继续上传或替换对应素材。
- 第 3/4 关可通过素材卡预览弹窗查看当前填写内容的整体呈现效果，预览不替代第 5 关报告。
- 上传图片在写入档案前压缩，目标大小不超过 500KB。
- 优先输出 WebP，浏览器不支持时回退 JPEG。
- 档案只保存压缩后的 DataURL 与元数据，不保存原始图片。

## QuickCheck

课时 2 的 QuickCheck 是自动过程记录，不是教师评分：

- T1：新闻与图片素材完成基础收集整理。
- T2：四关标准校准、个人素材复判、来源记录格式检查和三项自检完成。
- T3：新闻与图片各有 1 条有效初步疑点提示。

学生界面使用“体检完成情况”，避免出现“系统判定素材优秀”“来源真实可信”等误导表述。
QuickCheck 不在学生闯关页面显性展示评分；第 5 关只保留一个整体确认勾，系统在后台写入 T1/T2/T3 的 achieved、evidence 与过程 metrics。

## 阶段快照

课时 2 的阶段快照使用 `lesson2-full`，按第 1 关到第 5 关顺序导出当前过程数据、新闻/图片素材卡、实时 QuickCheck 结果和整体进度指针。
