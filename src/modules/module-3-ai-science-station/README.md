# Module 3 — AI 科学传播站

模块 3 是 ClassQuest 已完成并验证过的七年级课程模块。本轮平台化重构只改变目录和路由命名空间，不改变业务行为。

## 已完成范围

- 课时 1：项目启动与定题（5 关）
- 课时 2：证据采集与规范记录（5 关）
- 课时 3：素材整理与证据加工（5 关）
- 课时 4：结论形成与网页传播（5 关）
- 课时 5：预演展示与反馈优化（2 关）
- 课时 6：终版海报路演与表达设计（2 关）

合计 6 个课时、24 个关卡。

- 本地优先学习进度
- 继续学习包导出/导入

- 跨课时文件流转
- 阶段快照
- 教师演示模式
- Guard 与自动跳转行为
