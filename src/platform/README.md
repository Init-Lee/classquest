# Platform README

`src/platform/` 是 ClassQuest 的平台层。

## 职责

- 平台门户首页
- 全局路由组合
- 模块注册表
- 平台级外壳
- 旧路径重定向

## 不负责

- 模块 3 课时逻辑
- 模块 4 题卡逻辑
- 继续学习包格式
- 教师审核业务
- 后端 API 具体实现

## 当前路由

```text
/             → 平台门户
/module/3     → 模块 3 首页
/module/4     → 模块 4 占位首页
/lesson/*     → 重定向到 /module/3/lesson/*
```
